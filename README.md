Intro

Moincoin is a free open source project derived from Bitcoin.

MoinCoin is built on the foundation of Bitcoin, PPCoin, NovaCoin, and BitcoinDark, with a modified POS system.

Short: MOIN

Algorithm SHA256

RPC Port: 59902 P2P Port: 59901
